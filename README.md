# go-docker-demo

A simple Go web app containerized with Docker.

## How to Run Locally
```bash
go run main.go
```
## Docker Hub Image
You can pull the Docker image using:
```bash
docker pull ahmedd777/go-docker-demo:v1
```
## Docker Hub Link:
```bash
https://hub.docker.com/r/ahmedd777/go-docker-demo
```
